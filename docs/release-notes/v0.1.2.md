# AgentPrey v0.1.2

This release improves CLI ergonomics and refreshes release infrastructure/docs.

## Highlights

- Expanded top-level `agentprey -h` help text with clearer command descriptions
- Added practical scan/auth/vector usage examples directly in CLI help output
- Published crate `agentprey` v0.1.2 on crates.io
- Updated release workflow to use a dedicated release token secret (`GH_RELEASE_TOKEN`)

## Installation

```bash
cargo install agentprey --locked --force
```

## Verify

```bash
agentprey --version
agentprey --help
```
