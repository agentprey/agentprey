# AgentPrey v0.1.6

This release improves installation and release distribution without changing the
shipped scan surface.

## Highlights

- Added a public shell installer at `https://agentprey.com/install`
- Added SHA256 checksum sidecars for GitHub release archives
- Kept `cargo install agentprey --locked` as the cross-platform fallback path

## Install or upgrade

```bash
curl -fsSL https://agentprey.com/install | sh
```

## Install with Cargo

```bash
cargo install agentprey --locked --force
```

## Roll back to this exact version

```bash
cargo install agentprey --locked --version 0.1.6 --force
```

## Verify

```bash
agentprey --version
agentprey scan --help
agentprey init
```

## Notes

- The shell installer currently supports Linux x86_64 and Apple Silicon macOS.
- Intel Mac users should continue using `cargo install agentprey --locked` for now.
