# AgentPrey v0.1.0-beta.1

This is the first public beta release candidate.

## Highlights

- HTTP endpoint scanner with vector-driven prompt-injection coverage
- 20 curated prompt-injection vectors (direct, indirect, multi-turn)
- Terminal, JSON, and HTML reporting outputs
- Retry/backoff, rate limiting, and bounded concurrency controls
- Default-on response redaction for output artifacts
- Deterministic `scripts/beta_smoke.sh` baseline validation

## Installation

Download the archive for your platform from GitHub releases:

- `agentprey-x86_64-unknown-linux-gnu.tar.gz`
- `agentprey-aarch64-apple-darwin.tar.gz`
- `agentprey-x86_64-pc-windows-msvc.zip`

Each archive includes:

- `agentprey` binary
- `vectors/` dataset
- `scripts/mock_agent.py`
- `scripts/beta_smoke.sh`
- `README.md` and `CHANGELOG.md`

## Verification

Run the smoke harness after extraction:

```bash
bash scripts/beta_smoke.sh
```
