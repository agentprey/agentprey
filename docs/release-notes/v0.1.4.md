# AgentPrey v0.1.4

This release packages the current shipped CLI surface into a single patch release: OpenClaw local-path scanning, TUI mode, upload/share support, refreshed docs, and fuller release-candidate smoke coverage.

## Highlights

- Added OpenClaw local-path scanning with `agentprey scan --type openclaw --target ./path/to/project`
- Added TUI scan mode with `agentprey scan --ui tui`
- Added opt-in cloud upload with `--upload`, returning `scan_run_id`, `share_id`, and optional `share_url`
- Added public share pages for uploaded scan artifacts
- Added OpenClaw and release-candidate smoke scripts for launch hardening

## Install or upgrade

```bash
cargo install agentprey --locked --force
```

## Roll back to this exact version

```bash
cargo install agentprey --locked --version 0.1.4 --force
```

## Verify

```bash
agentprey --version
agentprey scan --help
```

## Notes

- OpenClaw scans use a checked-out local project path, not a URL.
- Upload is opt-in and no telemetry is sent when `--upload` is omitted.
- Share pages are public-by-link and artifact-driven; there is still no browser dashboard or MCP adapter in the shipped product.
